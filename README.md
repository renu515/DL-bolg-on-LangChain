# Deep Technical Blog on LangChain
This repository contains my internship assignment on LangChain, where I explored its architecture, components, and real-world applications.

##  About the Project
LangChain is a framework used to build applications powered by Large Language Models (LLMs).  
This project explains how LangChain helps in creating intelligent systems using modular components like prompts, chains, memory, and agents.
